  • Deploy your site:

    Divshot.com

    for easy CLI deployment


    1. Sign up for divshot static hosting (hangar)
    2. Select files you'd like to upload (CTRL + click)
    3. Right-click, "Compress files" (or "zip files")
    4. Create an "app" in Divshot
    5. Scroll down to the production environment and click "upload"
    6. File upload menu > select your zipped file (*.zip)
    7. Push the file online, then share your URL on Slack!
